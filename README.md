# Customer Churn Analysis and Prediction

This project analyzes customer churn data using machine learning techniques to predict which customers are at risk of leaving a service.

---

## 📊 Project Overview

**Objective**: To analyze key factors influencing customer churn and build a model to predict churners effectively.

**Techniques Used**:
- Data Cleaning & EDA
- Feature Engineering
- Machine Learning Modeling (Logistic Regression, Random Forest, etc.)
- Evaluation Metrics (Accuracy, Precision, Recall, F1 Score)
- Visualization using Matplotlib and Seaborn
